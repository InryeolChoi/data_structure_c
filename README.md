## 연결리스트
연결리스트에는 3가지 종류가 존재한다.
기본, 이중, 원형 연결리스트가 그것이다.

### 기본 연결리스트- 생성, 소멸, 연산 구현 
- 탐색, 삽입, 삭제, 갯수세기 구현
- 테스트 구현

### 이중 연결리스트
- 생성, 소멸, 연산 구현
- 탐색, 삽입, 삭제, 갯수세기 구현
- 테스트 구현

### 원형 연결리스트
- 생성, 소멸, 연산 구현
- 탐색, 삽입, 삭제, 갯수세기 구현
- 테스트 구현

## 스택
- 가장 마지막에 들어간 데이터가 제일 먼저 나오고 (LIFO) 가장 먼저 들어간 데이터는 가장 나중에 나옴. (FILO)
- 자동 메모리 (c언어에서 변수가 저장되는 곳) 역시 스택 구조로 이뤄져 있음.

### 배열로 스택 구현
* 전체 스택 = 구조체
    * 용량
    * 최상위 노드의 위치
    * 노드들 -> 이 부분이 배열
        * 동적할당으로 배열을 만든다.
        * 가리키는 것은 맨 아래의 노드
* 노드 한 개 = 구조체
    * 데이터

### 연결리스트로 스택 구현
* 전체 스택 = 구조체
    * 최상위 노드의 위치
    * 노드들 -> 이 부분이 연결리스트
    * 가리키는 것은 맨 아래의 노드
* 노드 한 개 = 구조체
    * 데이터
    * 다음 노드
